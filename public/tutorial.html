Como obter sua API do Google - Passo a Passo

1 Criar um Projeto no Google Cloud

Acesse o Google Cloud Console:

console.cloud.google.com
  1. Clique em "Selecionar projeto" no topo da página
  2. Clique em "Novo Projeto"
  3. Dê um nome ao projeto (ex: "SiloOrganizer")
  4. Clique em "Criar"

Nota: Se for sua primeira vez, você precisará aceitar os termos e pode precisar adicionar informações de cobrança (não será cobrado para uso básico).

2 Ativar a API Custom Search

Com o projeto selecionado:

  1. No menu lateral, clique em "APIs e Serviços" → "Biblioteca"
  2. Na barra de pesquisa, digite "Custom Search API"
  3. Clique no resultado "Custom Search API"
  4. Clique no botão azul "ATIVAR"
  5. Aguarde a ativação (cerca de 10 segundos)

3 Criar uma Chave de API

  1. No menu lateral, vá em "APIs e Serviços" → "Credenciais"
  2. Clique em "+ CRIAR CREDENCIAIS" no topo
  3. Selecione "Chave de API"
  4. Uma nova chave será criada e exibida
  5. COPIE E GUARDE esta chave! Você usará ela no SiloOrganizer

Dica: A chave se parece com isso: AIzaSyBsL7xxxxxx

4 Criar um Mecanismo de Pesquisa Personalizado

Agora vamos criar o mecanismo de busca:

Criar Mecanismo de Pesquisa
  1. Em "Sites para pesquisar", digite: *.com.br
  2. Em "Idioma", selecione "Português (Brasil)"
  3. Dê um nome (ex: "SiloOrganizer Brasil")
  4. Clique em "Criar"
  5. Na próxima tela, clique em "Personalizar"
  6. Ative a opção "Pesquisar em toda a web"
  7. Clique em "Configurações" → "Básico"
  8. COPIE o "ID do mecanismo de pesquisa"

Dica: O ID se parece com isso: c11b37b2xxxxx

5 Pronto! Configure suas credenciais

Agora você tem tudo que precisa:

  • API Key: Cole a chave da API do Google que você criou no passo 3
  • CSE ID: Cole o ID do mecanismo de pesquisa do passo 4
Voltar e Configurar Agora

Perguntas Frequentes

É gratuito?

Sim! O Google oferece 100 buscas gratuitas por dia. Perfeito para uso pessoal e pequenos projetos.

Preciso colocar cartão de crédito?

O Google pode pedir, mas não cobrará nada se você ficar dentro do limite gratuito.

Posso usar para buscar em qualquer site?

Sim! Ao ativar "Pesquisar em toda a web" no passo 4, você pode buscar em qualquer site.
